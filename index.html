Recipe-Book-App

This Project is a Recipe-Book-App created with Angular. The main part of the app was not the styling but more the functionalities. Splitting up the logic into different components and services and also using rxjs library/ observerables for better handling of asynchronous dataflow/events.
Features of the app: The User can log in via Firebase authentification, create Recipes, add the needed ingredients into a shopping list, fetch recipes from firebase and save them also. The app is hosted with Firebase.

Crown-K

This Project is a E-Commerce Store created with React(only Hooks and Context API). As you can see in my GitHub repository crown-clothing I also used redux in the project but for this small project there was not such a big benefit as for much larger apps. Context API is perfect for smaller Apps like Crown-K because it gives the user the same functionality without hurting readability. If the App would become bigger I would use redux.
Features of the app: The User can log in via Firebase authentification, select items, put them into shopping cart and buy them(demo only) via the Stripe API. The app is hosted with Heroku.
